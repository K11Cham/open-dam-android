# open-dam-android
A local Gambian board game, but open source, and written in React Native for Android.

# Story

I lived in The Gambia for about 8 years. During those 8 years, I'd travel to my father's home village, Kerr Jarga Jobe, in the North Bank Region.
There, one of the most common traditional games played by both children and adults was Dam.
There were many variations, but the most common was the classic 5x5 layout for 2 players, drawn in the sand and using sticks and stones as pieces.

I decided I'd reconstruct these memories, by creating a digital version of the game, so that I can play it at anytime, and share Gambian culture with anyone else who gets to experience it.

**Fun fact:** This Android version was unironically made first, before the web version. Sometimes you just gotta start with what you have in your pocket.
